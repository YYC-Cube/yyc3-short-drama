# 117-YYC3-Short-Drama-测试验证-功能测试用例

## 概述

本文档详细描述YYC3-Short-Drama项目核心功能的测试用例，包含前置条件、执行步骤和预期结果，确保所有功能按设计要求正常工作。

## 核心内容

### 1. 用户管理模块测试用例

#### 1.1 用户注册功能

**TC001 - 验证正常用户注册流程**
- **前置条件**:
  - 系统正常运行
  - 访问注册页面
- **执行步骤**:
  1. 输入有效的邮箱地址
  2. 输入符合要求的密码（至少8位，包含字母和数字）
  3. 输入用户名
  4. 点击"注册"按钮
  5. 检查邮箱是否收到验证邮件
  6. 点击验证链接
- **预期结果**:
  - 注册请求成功提交
  - 邮箱收到验证邮件
  - 账户激活成功
  - 跳转到登录页面

**TC002 - 验证重复邮箱注册**
- **前置条件**:
  - 系统正常运行
  - 已存在使用相同邮箱的账户
- **执行步骤**:
  1. 输入已存在的邮箱地址
  2. 输入其他有效信息
  3. 点击"注册"按钮
- **预期结果**:
  - 显示错误提示"邮箱已被注册"
  - 注册失败

**TC003 - 验证无效邮箱格式**
- **前置条件**:
  - 系统正常运行
  - 访问注册页面
- **执行步骤**:
  1. 输入无效邮箱格式（如：invalid-email）
  2. 输入其他有效信息
  3. 点击"注册"按钮
- **预期结果**:
  - 显示错误提示"请输入有效的邮箱地址"
  - 注册失败

**TC004 - 验证弱密码注册**
- **前置条件**:
  - 系统正常运行
  - 访问注册页面
- **执行步骤**:
  1. 输入有效邮箱
  2. 输入弱密码（如：123456）
  3. 输入用户名
  4. 点击"注册"按钮
- **预期结果**:
  - 显示错误提示"密码强度不够，请使用至少8位包含字母和数字的密码"
  - 注册失败

#### 1.2 用户登录功能

**TC005 - 验证正常用户登录**
- **前置条件**:
  - 用户已注册并通过邮箱验证
  - 访问登录页面
- **执行步骤**:
  1. 输入注册时使用的邮箱
  2. 输入正确的密码
  3. 点击"登录"按钮
- **预期结果**:
  - 登录成功
  - 跳转到首页或之前访问的页面
  - 显示用户信息和个性化内容

**TC006 - 验证错误密码登录**
- **前置条件**:
  - 用户已注册并通过邮箱验证
  - 访问登录页面
- **执行步骤**:
  1. 输入正确的邮箱
  2. 输入错误的密码
  3. 点击"登录"按钮
- **预期结果**:
  - 显示错误提示"邮箱或密码错误"
  - 登录失败
  - 保持在登录页面

**TC007 - 验证未验证邮箱登录**
- **前置条件**:
  - 用户已注册但未通过邮箱验证
  - 访问登录页面
- **执行步骤**:
  1. 输入未验证的邮箱
  2. 输入正确的密码
  3. 点击"登录"按钮
- **预期结果**:
  - 显示提示"请先验证您的邮箱地址"
  - 提供重新发送验证邮件的选项
  - 登录失败

#### 1.3 用户信息管理功能

**TC008 - 验证用户信息修改**
- **前置条件**:
  - 用户已登录
  - 访问个人资料页面
- **执行步骤**:
  1. 修改用户名
  2. 修改个人简介
  3. 上传新的头像图片
  4. 点击"保存"按钮
- **预期结果**:
  - 信息修改成功
  - 显示成功提示
  - 页面显示更新后的信息

**TC009 - 验证密码修改**
- **前置条件**:
  - 用户已登录
  - 访问密码修改页面
- **执行步骤**:
  1. 输入当前密码
  2. 输入新密码
  3. 确认新密码
  4. 点击"修改密码"按钮
- **预期结果**:
  - 密码修改成功
  - 显示成功提示
  - 需要重新登录

  2. 点击"已关注"按钮
  3. 确认取消关注操作
- **预期结果**:
  - 取消关注成功
  - 按钮状态变回"关注"
  - 用户A的关注列表移除用户B
  - 用户B的粉丝列表移除用户A
