# 172-YYC3-Short-Drama-产品文档-产品功能说明书

## 概述

本文档详细描述YYC3-Short-Drama项目所有功能模块、使用流程和操作规范，为用户提供完整的使用指南。

## 核心内容

### 1. 用户账户管理

#### 1.1 用户注册

**功能描述**: 新用户通过邮箱注册账户，完成邮箱验证后激活账户。

**操作步骤**:
1. 访问注册页面 `/register`
2. 填写注册信息：
   - 邮箱地址（必填）
   - 密码（必填，至少8位，包含字母和数字）
   - 确认密码（必填）
   - 用户名（必填）
   - 手机号（选填）
3. 阅读并同意《用户协议》和《隐私政策》
4. 点击"注册"按钮
5. 检查邮箱，点击验证链接
6. 验证成功后自动跳转到登录页面

**注意事项**:
- 邮箱必须是真实有效的邮箱地址
- 密码强度需满足系统要求
- 验证链接有效期为24小时

#### 1.2 用户登录

**功能描述**: 已注册用户通过邮箱和密码登录系统。

**操作步骤**:
1. 访问登录页面 `/login`
2. 输入注册时使用的邮箱地址
3. 输入密码
4. 可选择"记住我"选项（7天内免登录）
5. 点击"登录"按钮
6. 登录成功后跳转到首页

**注意事项**:
- 连续5次登录失败将锁定账户30分钟
- 支持使用第三方账号登录（微信、QQ等）

#### 1.3 密码重置

**功能描述**: 用户忘记密码时，通过邮箱重置密码。

**操作步骤**:
1. 在登录页面点击"忘记密码"
2. 输入注册时使用的邮箱地址
3. 点击"发送重置链接"按钮
4. 检查邮箱，点击重置链接
5. 设置新密码
6. 确认新密码
7. 重置成功后可使用新密码登录

**注意事项**:
- 重置链接有效期为1小时
- 新密码不能与旧密码相同

#### 1.4 个人信息管理

**功能描述**: 用户可以查看和修改个人信息。

**操作步骤**:
1. 登录后进入个人中心
2. 点击"个人信息"
3. 修改可编辑的信息：
   - 昵称
   - 头像
   - 个人简介
   - 手机号
   - 生日
   - 性别
4. 点击"保存"按钮

**注意事项**:
- 邮箱地址修改需要验证
- 头像上传限制为2MB以内

### 2. AI智慧编剧功能

#### 2.1 八卦剧本生成器

**功能描述**: 基于河图洛书理论的剧本结构生成，AI辅助用户创作剧本。

**操作步骤**:
1. 进入"AI编剧"模块
2. 选择"八卦剧本生成器"
3. 填写剧本参数：
   - 剧本主题（必填）
   - 剧本类型（古装、现代、科幻等）
   - 主要角色数量
   - 故事长度（短篇、中篇、长篇）
   - 风格偏好（喜剧、悲剧、正剧等）
4. 点击"生成剧本"按钮
5. 等待AI生成（约30-60秒）
6. 查看生成的剧本大纲
7. 可选择继续生成详细剧本或修改大纲
8. 保存剧本到个人作品

**注意事项**:
- 每日免费生成次数为3次
- 生成次数可通过星币购买
- 生成内容建议人工审核后使用

#### 2.2 古今对话转换

**功能描述**: 将现代台词转换为古风文言文，提升剧本文化韵味。

**操作步骤**:
1. 进入"AI编剧"模块
2. 选择"古今对话转换"
3. 输入或粘贴现代台词
4. 选择转换风格：
   - 古典文言
   - 半文半白
   - 通俗古风
5. 点击"转换"按钮
6. 查看转换结果
7. 可选择"重新转换"或"复制结果"
8. 支持批量转换（最多10条）

**注意事项**:
- 转换结果仅供参考，建议人工调整
- 支持自定义词库添加

#### 2.3 文化元素库

**功能描述**: 提供河洛文化相关素材，辅助剧本创作。

**操作步骤**:
1. 进入"AI编剧"模块
3. 选择"文化元素库"
4. 浏览或搜索文化元素：
   - 历史人物
   - 文化事件
   - 地理地点
   - 民俗习惯
   - 典故传说
5. 点击元素查看详细信息
6. 可选择"收藏"或"使用"
7. 支持用户贡献新元素

**注意事项**:
- 用户贡献的元素需要审核后显示
- 收藏的元素可在创作时快速引用

**注意事项**:

#### 3.2 智能学习系统

**操作步骤**:

2. 选择"智能学习"

5. 完成学习内容后获得星币奖励

### 4. 虚实共生体系功能

#### 4.1 VR文化体验

**功能描述**: 沉浸式历史场景重现，互动式文化探索。

**操作步骤**:
1. 进入"虚实共生"模块
2. 选择"VR体验"
3. 选择体验场景：
   - 洛阳古城
   - 龙门石窟
   - 白马寺
   - 牡丹花会
4. 进入VR场景
5. 使用VR设备进行互动：
   - 移动和旋转视角
   - 点击交互点查看信息
   - 与其他用户在线互动
6. 退出体验后可分享体验记录

**注意事项**:
- 需要VR设备支持
- 支持多人在线协作体验

#### 4.2 AR增强现实

**功能描述**: 实景文化信息叠加，扫描识别文化元素。

**操作步骤**:
1. 进入"虚实共生"模块
2. 选择"AR体验"
3. 使用手机摄像头扫描：
   - 文化建筑
   - 文物展品
   - 文化标识
4. 查看AR叠加的文化信息
5. 可收藏或分享识别结果

**注意事项**:
- 需要摄像头权限
- 建议在光线充足环境下使用

### 5. 星值经济体系功能

#### 5.1 星币获取

**功能描述**: 用户通过多种方式获取星币。

**获取方式**:
- **内容创作奖励**: 发布剧本获得10-100星币
- **学习成就奖励**: 完成学习内容获得5-50星币
- **社区贡献奖励**: 贡献文化元素获得1-10星币
- **活动参与奖励**: 参与平台活动获得20-200星币
- **每日签到**: 每日签到获得5星币
- **邀请好友**: 邀请好友注册获得50星币

**操作步骤**:
1. 进入"星币中心"
2. 查看星币余额和获取记录
3. 了解各种获取方式
4. 参与相应活动获取星币

#### 5.2 星币消费

**功能描述**: 使用星币购买虚拟商品和服务。

**消费场景**:
- **高级功能解锁**: 解锁AI高级功能（50-500星币）
- **虚拟物品购买**: 购买虚拟道具和装饰（10-200星币）
- **内容打赏**: 打赏优秀作品（1-100星币）
- **会员服务**: 购买会员服务（100-1000星币）

**操作步骤**:
1. 进入"星币中心"
2. 选择"星币商城"
3. 浏览虚拟商品
4. 点击商品查看详情
5. 点击"购买"按钮
6. 确认购买
7. 购买成功后商品自动到账

#### 5.3 星币充值

**功能描述**: 通过多种支付方式充值星币。

**充值方式**:
- 微信支付
- 支付宝
- 银行卡
- PayPal

**充值比例**: 1元人民币 = 10星币

**操作步骤**:
1. 进入"星币中心"
2. 选择"充值"
3. 选择充值金额：
   - 10元 = 100星币
   - 50元 = 500星币

4. 选择支付方式
5. 扫码或跳转支付
6. 完成支付后星币即时到账

### 6. 内容管理功能

#### 6.1 剧本创作

**功能描述**: 用户可以创建、编辑和发布剧本。

**操作步骤**:
1. 进入"创作中心"
2. 点击"新建剧本"
3. 填写剧本信息：
   - 剧本标题（必填）
   - 剧本简介（必填）
   - 剧本类型（必选）
   - 剧本标签（必选）
4. 使用编辑器编写剧本内容：
   - 支持富文本编辑
   - 支持插入图片和视频
   - 支持多人协作编辑
5. 点击"保存草稿"
6. 完成后点击"发布"
7. 等待审核通过后公开

**注意事项**:
- 剧本内容需符合平台规范
- 审核时间通常为1-3个工作日

**功能描述**: 浏览平台上的各类文化内容。

### 8. 常见问题

#### 8.1 账户相关

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，通过邮箱重置密码。

**Q: 如何修改邮箱？**
A: 在个人中心修改邮箱，需要验证新邮箱地址。

**Q: 账户被锁定怎么办？**
A: 等待30分钟后自动解锁，或联系客服解锁。

#### 8.2 功能相关

**Q: AI生成次数用完了怎么办？**
A: 可以使用星币购买额外生成次数。

**Q: 剧本审核需要多久？**
A: 通常为1-3个工作日，节假日可能延长。

**Q: VR体验需要什么设备？**
A: 支持主流VR设备，如Oculus、HTC Vive等。

#### 8.3 星币相关

**Q: 星币可以提现吗？**
A: 星币仅限平台内使用，不支持提现。

**Q: 星币有有效期吗？**
A: 星币永久有效，无使用期限。

**Q: 充值失败怎么办？**
A: 请检查支付方式是否正常，或联系客服处理。
