# 生成标题接口

## 接口描述

使用 AI 为剧本生成吸引人的标题。

## 请求信息

### 请求方法

- POST

### 请求路径

- `/api/ai-script/titles`

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|-------|------|------|------|
| `scriptContent` | string | 否 | 剧本内容，用于生成相关标题 |
| `theme` | string | 否 | 剧本主题，当没有提供剧本内容时使用 |
| `style` | string | 否 | 标题风格，如：文艺、悬疑、温情等，默认 "综合" |
| `count` | number | 否 | 生成标题数量，默认 5 |

### 请求示例

```json
{
  "scriptContent": "标题：母爱的温度\n人物：母亲、儿子\n场景：现代都市家庭\n剧本正文：...",
  "style": "温情",
  "count": 5
}
```

## 响应信息

### 成功响应

**状态码**: 200 OK

```json
{
  "success": true,
  "message": "标题生成成功",
  "data": {
    "titles": [
      "母爱的温度",
      "家的温暖",
      "亲情的力量",
      "爱的守护",
      "温暖的港湾"
    ],
    "style": "温情",
    "count": 5
  }
}
```

### 错误响应

**状态码**: 400 Bad Request

```json
{
  "success": false,
  "message": "请提供剧本内容或主题"
}
```

**状态码**: 401 Unauthorized

```json
{
  "success": false,
  "message": "请先登录"
}
```

**状态码**: 500 Internal Server Error

```json
{
  "success": false,
  "message": "标题生成失败，请稍后重试"
}
```

## 接口说明

- 该接口需要用户已登录，否则会返回 401 错误
- 标题生成过程通常很快，几秒钟内完成
- 生成的标题会与剧本内容或主题相关
- 可以指定标题风格以获得不同类型的标题

## 使用场景

- 为剧本寻找合适的标题
- 为已有剧本生成多个标题选项
- 基于主题生成创意标题

## 注意事项

- 生成的标题仅供参考，可以根据需要进行修改
- 请确保剧本内容或主题符合相关法律法规
