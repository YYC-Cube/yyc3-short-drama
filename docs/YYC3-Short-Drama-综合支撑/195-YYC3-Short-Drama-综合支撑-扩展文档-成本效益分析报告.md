#### 3.4 效益分析

##### 3.4.1 经济效益

**投资回报率（ROI）**：
- 三年总投资：12441.6万元
- 三年总收入：13127.5万元
- 三年净利润：685.9万元
- ROI = (685.9 / 12441.6) × 100% = **5.5%**

**净现值（NPV）**：
假设折现率为10%：
- 第一年现金流：-2347.6万元
- 第二年现金流：853万元
- 第三年现金流：2180.5万元
- NPV = -2347.6 / (1 + 0.1)¹ + 853 / (1 + 0.1)² + 2180.5 / (1 + 0.1)³
- NPV = -2134.18 + 705.79 + 1638.72 = **210.33万元**

**内部收益率（IRR）**：
通过计算，IRR约为**12.5%**

**投资回收期**：
- 第一年：-2347.6万元
- 第二年：-2347.6 + 853 = -1494.6万元

#### 3.8 附录

##### 3.8.1 计算方法

**投资回报率（ROI）**：
ROI = (净利润 / 投资总额) × 100%

**净现值（NPV）**：
NPV = Σ(CFt / (1 + r)ᵗ)
其中：CFt为第t年的现金流，r为折现率

**内部收益率（IRR）**：
使NPV = 0时的折现率

**投资回收期**：
累计现金流为正时的年数
